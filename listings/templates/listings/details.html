Construction Model Details

Technical overview of the Artificial Neural Network used for construction cost prediction.

  • Algorithm: Artificial Neural Network (ANN)
  • Model Type: Deep Learning – Cost Prediction
  • Input Features:
    State City Built-up Area (sq.ft) Number of Bedrooms (BHK) & Bathrooms Age of Building
  • Output: Estimated Construction Cost (₹ in Lakhs)
  • Deep Learning Framework: TensorFlow / Keras
  • Backend Integration: Django (Model loaded using Joblib)
  • Model Storage: /model/construction_ann_pipeline.joblib
  • Processing Flow: Input → Preprocessing → ANN Hidden Layers → Cost Prediction
